Add a `span` with an icon applied.

```js static
import { Dashicon, Dashicons } from 'wp-admin-ui-react';

const Icon = () => <Dashicon dashicon="thumbs-up" />;
const Icon2 = () => <Dashicon dashicon={Dashicons.wordpress} />;
```
