# Dashicon

Add a Dashicon icon to your UI.

## Usage

```jsx
import { Dashicon } from 'wp-admin-ui-react';

const TrashButton = () => (
  <button>
    <Dashicon dashicon="trash" />
  </button>
);
```

## Props

Props available on the `<Dashicon />`-component:

| Name       | Required | Default | Description             |
| :--------- | :------- | :------ | :---------------------- |
| `dashicon` | x        | -       | Name of dashicon to use |
